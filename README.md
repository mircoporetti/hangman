# Hangman

A hangman game built with Elixir and Phoenix LiveView.

Guess the hidden word one letter at a time. Each wrong guess brings you closer to losing. You have 7 attempts before the game is over. Guess all the letters to win.

## Setup

  * Run `mix setup` to install and setup dependencies
  * Start Phoenix endpoint with `mix phx.server` or inside IEx with `iex -S mix phx.server`

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.

## Running tests

```bash
mix test
```
